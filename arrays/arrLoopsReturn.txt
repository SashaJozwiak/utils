| Методы перебора массива | Возвращает |
| ----------------------- | ---------- |
| forEach()               | undefined  |
| map()                   | новый массив |
| filter()                | новый массив |
| reduce()                | одно значение |
| every()                 | true/false |
| some()                  | true/false |
| find()                  | один элемент массива |
| findIndex()             | индекс искомого элемента в массиве |
